# thingsboard-scripts

Contents:
* device
  * echo attribute updates
  * RPC demo
* export
  * get attribute per device name (from devices in a csv list)
  * download timeseries of all devices given a device profile

## Run

```bash
pip install -r requirements.txt
python main.py
```

If you have `uv` installed, see [CONTRIBUTING.md](./CONTRIBUTING.md).
